# Signal Processing

This repository contains basic implementations of digital signal processing concepts using Python.

## Contents
- **FFT Analysis**
  - Frequency spectrum analysis of a signal using Fast Fourier Transform (FFT).

- **Digital Filters**
  - Implementation of Low-Pass, High-Pass, and Band-Pass Butterworth filters.

- **Frequency Analysis Before & After Filtering**
  - Comparison of frequency components of a noisy signal before and after band-pass filtering.

## Tools & Libraries
- Python
- NumPy
- SciPy
- Matplotlib

## Usage
Run each Python file individually to visualize the results.
